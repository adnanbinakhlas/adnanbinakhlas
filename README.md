## 🚀 Featured Project: RentNest

**RentNest** is a comprehensive rental property management platform engineered specifically for the Bangladesh market.

### Technical Implementation & Highlights

- **Scalable REST API:** Built a robust backend architecture using **Express.js**, **PostgreSQL**, and **Prisma ORM**.
- **Security & Auth:** Implemented strict authentication and role-based authorization using **JWT** with the `jose` library.
- **Performance Optimization:** Drastically reduced database load and latency by integrating **Redis caching**.
- **Data Management:** Developed advanced query mechanisms including full-text search, filtering, sorting, pagination, and dynamic field selection.
- **Media & Payments:** Integrated **Cloudinary** for optimized image storage pipelines and **Stripe** for secure subscription processing.
- **Data Integrity:** Applied rigorous runtime type-checking and request payload validation utilizing **Zod**.

🔗 **Repository:** [github.com/adnanbinakhlas/rent-nest-server](https://github.com/adnanbinakhlas/rent-nest-server)
